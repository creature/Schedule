Schedule/badge management site for during Electromagnetic Field camp.

Starting
========
```
sudo apt-get install python-virtualenv python-dev sqlite3
make init
make update
make data
```

For live (or if you prefer):
```
sudo apt-get install postgresql
```

Running
=======
```
make update
make data
make
```
